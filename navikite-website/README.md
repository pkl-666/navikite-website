# NaviKite website

Static website for navikite.com.

## Files

- `index.html` - main page
- `styles.css` - visual design
- `script.js` - language toggle
- `CNAME` - GitHub Pages custom domain
- `assets/` - logo, app icon and PDF guide

## GitHub Pages

Publish from the repository root on the `main` branch, then set the custom domain to:

```text
navikite.com
```
